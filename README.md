# GoPad

A compact, high-performance controller powered by the Seeed XIAO RP2040, specifically engineered for seamless QLab integration and theater production workflows. This build features five tactile keys and an integrated backlight to ensure precision and visibility in dark booth environments.

QLab Functionality:
Tactile Cues: Dedicated physical buttons for GO, Stop, Next, and Previous.

Low-Light Design: Customizable backlighting allows for easy operation even in dark environments.

Plug & Play: Emulates a standard HID keyboard for instant compatibility with macOS and qLab.
